This plugin adds a category search to the Quick Edit menu in WordPress.

== Description ==

With the Quick Edit Category Search plugin, you can quickly search and filter categories in the quick edit menu of posts. This allows you to quickly find a specific category even on a site with a large number of categories.

![screenshot](https://wordpress.org/plugins/quick-edit-category-search/screenshot-1.png)

== Installation ==

1. Upload the `quick-edit-category-search` folder to your plugin directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

1. Navigate to the posts list page.
2. Click 'Quick Edit' on the post you want to edit.
3. Enter the category name in the search box added by the plugin.
4. The category checklist is dynamically updated based on the search results.
